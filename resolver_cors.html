Como Resolver o Erro CORS

Erro: Access to internal resource at 'file:///C:/RPJDigitalSites/SALARIO/manifest.json' from origin 'null' has been blocked by CORS policy

Por que isso acontece?

Este erro ocorre porque você está acessando os arquivos diretamente pelo sistema de arquivos (protocolo file://) em vez de usar um servidor web (protocolo http://).

Os navegadores modernos bloqueiam o acesso a recursos como o manifest.json quando abertos diretamente do sistema de arquivos por questões de segurança.

Como resolver

Para resolver este problema, você precisa servir seus arquivos através de um servidor HTTP:

Solução 1: Usar o servidor Node.js incluído

  1. Abra o Prompt de Comando (CMD) ou PowerShell
  2. Navegue até a pasta do seu projeto:
    cd C:\RPJDigitalSites\SALARIO
  3. Execute o servidor Node.js:
    node server.js
  4. Acesse seu aplicativo no navegador através de:
    http://localhost:8080

Solução 2: Usar extensões de navegador

Para desenvolvimento rápido, você pode usar extensões que desativam temporariamente as restrições de CORS:

  • Para Chrome: CORS Unblock
  • Para Firefox: CORS Everywhere
Importante: Para que seu PWA funcione corretamente em produção, você deve hospedá-lo em um servidor web real (como GitHub Pages, Netlify, Vercel ou qualquer hospedagem web).

Solução 3: Hospedagem web

Para uma solução permanente, hospede seu PWA em um serviço gratuito como:

  • GitHub Pages
  • Netlify
  • Vercel
  • Firebase Hosting
